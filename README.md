# Svelte Portfolio 🔥
Portfolio template made by using Svelte.

- Easy to configure.
- Dark/Light modes.

## Installation ✨
Clone this repository.
```bash
git clone https://github.com/s3h4n/Svelte-Portfolio.git
```
Install using dependencies.
```bash
npm install
```

## Usage 📚
Run following command to start the development server.
```bash
npm run dev
```
